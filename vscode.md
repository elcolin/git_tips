# VSCODE

## Shortcuts Linux
### Search occurrences in all files
    CTRL + SHIFT + F
### Search occurrences in file
    CTRL + F
### Select all ocurrences in said file
    CTRL + SHIFT + L
### Copy line
    CTRL + C
### Cut line
    CTRL + X
### Move line
    ALT + <Top arrow>/<Bottom arrow>
### Add empty line above current line
    CTRL + SHIFT + ENTER
